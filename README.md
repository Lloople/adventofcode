# adventofcode
My solutions for https://adventofcode.com

## Installation

```
composer install
```

## Usage

### Single day:

```
php day.php {day} {year}
```

> Default to the current day and current year

If you want to run a specific part of a day:

```
php day.php {day} {year} {part}
```

### Run a whole year:

```
php year.php {year}
```

> Default to the current year
